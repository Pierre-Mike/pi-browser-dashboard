# routes

TanStack Router file-based routes. `routeTree.gen.ts` is auto-generated by `@tanstack/router-plugin` — never edit by hand.

- `__root.tsx` — app shell: sticky header (logo + `⇧⇧ to jump` hint), `<Sidebar />`, `<Outlet />`, `<PaletteController />` mounted globally.
- `index.tsx` — `/` landing. Renders `<ProjectGrid />` with sessions + projects; handles loading / error / empty.
- `projects.$id.tsx` — `/projects/$id`. Looks up the project by id from `useProjects()`, renders `<ProjectDashboard />`.
- `sessions.$id.tsx` — `/sessions/$id` drill-in: header, transcript via `<TranscriptView />`, `<ChatComposer />`, `<TerminalTab />` tab.
